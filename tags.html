Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bolded text <strong>technical sophistication</strong> technical sophistication
a anchor creates hyperlinks <a href="page.html">technical sophistication</a> technical sophistication
img image adds an image <img src=twitter_logo.jpg>
code code montone text <code>technical sophistication</code> technical sophistication
header header creates a header <header>technical sophistication</header>
technical sophistication
div divider creates a block divide on the page <div>technical sophistication</div>
technical sophistication
span span creates an inline divide on the page <span>technical sophistication</span> technical sophistication
ol ordered list creates a numbered list <ol><li>technical sophistication</li></ol>
  1. technical sophistication
ul unordered list creates a bulleted list <ul><li>technical sophistication</li></ul>
  • technical sophistication
li list data lists contents of a list <li>technical sophistication</li>
  • technical sophistication
